欢迎来到《气候模拟》演示程序托管系统

《气候模拟》（北京大学，大气与海洋科学系）课程涉及到的演示程序和作业相关程序，整理在此github.com网站，以便选课同学方便地获取相关程序demo作为参考。我们会把不断修改这些程序，以确保bug最少的demo不断更新至此。相信这样会有利于选课同学的学习和课程质量的提高。

获取代码

任何时候，您都可以通过如下方式，获取本项目的最新版代码：

在Unix/Linux系统中，使用Git工具获取完整文件树，在命令行中输入：

$ git clone https://github.com/wensir365/climatemodeling.git

在Windows系统中，使用任意浏览器，从如下页面的右上角获取Zip或Tar包： https://github.com/wensir365/climatemodeling

我们的代码遵守GNU/GPL协议，希望所有下载、使用这些demo的用户，仍继续遵守GPL协议。
