CLIMA/DATA
This subdirectory has the data files needed by the climate code.

- CO2_tables.pdat
Data to calculate the moist adiabat in a CO2 atmosphere

- H2O_tables.pdat
Data to calculate the H2O moist adiabat 

- fluxesKGF_surf.pdat
Stellar fluxes from  an F2V, K2V and a G2V stars. 
Read by the subroutine choose_star.f if other star is choosen instead 
of the Sun.

- ir_expsums.pdat      
IR exponential sums for CO2, CH4 and H2O used in the old IR 
code.

- nearIR_expsums.pdat
Exponential sums for CO2, H2O and methane used by the solar code

- solar_data_38.pdat
Solar flux and data needed by the solar code
